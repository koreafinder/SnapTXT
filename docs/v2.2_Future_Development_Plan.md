### **5. 고급 Scope Guard (Enum 기반)** 🛡️

**목표**: v2.1.3 상수화를 넘어선 완전한 타입 안전성

**현재 (v2.1.3)**:
```python
STAGE2 = "S2"  # 상수 하드코딩
STAGE3 = "S3"
```

**v2.2 목표**:
```python
from enum import Enum, auto

class StageIdentifier(Enum):
    STAGE2 = auto()
    STAGE3 = auto()
    
class ScopePolicy(Enum):
    STAGE2_ONLY = auto()
    STAGE3_ONLY = auto()  
    COOPERATIVE = auto()
    FORBIDDEN = auto()

# 완전한 타입 안전성
def should_apply_rule(stage: StageIdentifier, ...) -> bool:
    policy = get_policy(phenomenon, stage)
    if policy == ScopePolicy.STAGE2_ONLY and stage == StageIdentifier.STAGE2:
        return True
```

**장점**:
- IDE 자동완성 지원
- 타입 체커 검증 가능  
- 런타임 값 오류 원천 차단
