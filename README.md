# SnapTXT 📸➡️📝🔊

**이미지에서 텍스트 추출 후 음성으로 듣는 학습 도구**

스크린샷이나 이미지 파일에서 한국어/영어 텍스트를 추출하고, 웹 브라우저에서 음성으로 들을 수 있는 웹 애플리케이션입니다.

## ✨ 주요 기능

- 🔍 **EasyOCR 기반 고정확도 텍스트 인식**: 한국어와 영어 텍스트를 정확하게 추출
- 📱 **드래그 앤 드롭 업로드**: 간편한 이미지 업로드 인터페이스
- 🔊 **웹 브라우저 TTS**: 추출된 텍스트를 음성으로 재생 (모바일 호환)
- 📋 **텍스트 복사**: 추출된 텍스트를 클립보드에 복사
- 📱 **모바일 최적화**: iPhone Safari 등 모바일 브라우저 지원

## 🚀 실행 방법

### 1. 의존성 설치
```bash
# 필요한 패키지 설치 (전역 설치 권장)
pip install flask Pillow easyocr torch torchvision opencv-python
```

### 2. 서버 실행
```bash
python main.py
```

### 3. 웹 브라우저에서 접속
```
http://127.0.0.1:5000
```

## 🛠 기술 스택

- **Backend**: Python, Flask
- **OCR Engine**: EasyOCR (한국어/영어 지원)
- **Frontend**: HTML5, CSS3, JavaScript
- **TTS**: Web Speech API
- **Image Processing**: PIL, OpenCV

## 📋 사용법

1. 웹 페이지에서 이미지 파일을 드래그 앤 드롭하거나 파일 선택
2. 자동으로 텍스트 추출 진행
3. 추출된 텍스트가 화면에 표시
4. "텍스트 읽기" 버튼으로 음성 재생
5. "텍스트 복사" 버튼으로 클립보드에 복사

## 💡 활용 예시

- 📚 **학습**: 교재나 문서 이미지를 텍스트로 변환 후 음성으로 학습
- 📖 **독서**: 책 페이지를 촬영하여 음성으로 듣기
- 📑 **문서 처리**: 스캔된 문서의 텍스트 추출
- 🌐 **번역**: 외국어 텍스트 추출 후 번역 도구 활용

## ⚙️ 시스템 요구사항

- Python 3.7+
- Windows/macOS/Linux
- 최소 4GB RAM (EasyOCR 모델 로딩용)
- 인터넷 연결 (최초 모델 다운로드)

## 🔧 설정 정보

- **지원 이미지 형식**: PNG, JPG, JPEG, GIF, BMP, TIFF
- **지원 언어**: 한국어(ko), 영어(en)
- **최대 파일 크기**: 16MB
- **기본 포트**: 5000
